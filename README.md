# DeepFuck Chat - 复古风格 AI 对话应用

一个基于 Vue 3 + TypeScript + Tailwind CSS 构建的复古风格 AI 聊天应用，支持与 DeepSeek API 进行智能对话。

## 🎨 项目特色

-   **复古像素风格 UI** - 采用 80 年代复古设计风格，带有像素化边框和阴影效果
-   **多会话管理** - 支持创建、切换、删除多个对话会话
-   **流式响应** - 实时显示 AI 回复内容，提供流畅的对话体验
-   **本地存储** - 自动保存对话记录和配置到浏览器本地存储
-   **灵活配置** - 支持自定义 API 密钥、模型、温度等参数
-   **多种系统提示词** - 内置多种对话风格（默认、编程、创意、学术、休闲）
-   **响应式设计** - 适配桌面和移动设备

## 🚀 技术栈

### 前端框架

-   **Vue 3** - 采用 Composition API 的现代 Vue 框架
-   **TypeScript** - 提供类型安全和更好的开发体验
-   **Vite** - 快速的构建工具和开发服务器

### 状态管理

-   **Pinia** - Vue 3 官方推荐的状态管理库
-   **@vueuse/core** - Vue 组合式 API 工具集

### 样式和 UI

-   **Tailwind CSS** - 实用优先的 CSS 框架
-   **PostCSS** - CSS 后处理器
-   **自定义复古主题** - 包含复古色彩和动画效果

### API 集成

-   **OpenAI SDK** - 用于与 AI API 通信
-   **DeepSeek API** - 默认集成 DeepSeek 聊天模型

## 🎯 核心功能

### 1. 会话管理

-   **新建对话** - 点击"START NEW CHAT"创建新的对话会话
-   **会话切换** - 在左侧会话列表中点击切换不同对话
-   **会话删除** - 鼠标悬停在会话上显示删除按钮
-   **自动标题** - 根据首条消息自动生成会话标题
-   **时间显示** - 显示会话的最后更新时间

### 2. 智能对话

-   **流式响应** - AI 回复实时显示，无需等待完整响应
-   **消息历史** - 保持完整的对话上下文
-   **加载状态** - 显示打字动画和加载提示
-   **错误处理** - 网络错误时显示友好提示

### 3. 配置管理

-   **API 配置** - 自定义 API 密钥、基础 URL、模型名称
-   **参数调节** - 调整温度参数控制回复随机性
-   **系统提示词** - 选择不同的对话风格和角色设定
-   **配置验证** - 实时验证配置参数的有效性

### 4. 数据持久化

-   **本地存储** - 所有对话和配置自动保存到浏览器
-   **数据恢复** - 刷新页面后自动恢复之前的对话
-   **配置重置** - 支持重置 API 配置或清空所有数据

## 🎨 UI 设计特色

### 复古风格元素

-   **像素字体** - 使用 Courier New 等等宽字体
-   **复古色彩** - 明亮的黄色、粉色、蓝色、绿色等 80 年代色彩
-   **立体边框** - 4px 黑色边框配合阴影效果
-   **按钮设计** - 立体按钮带有悬停效果

### 动画效果

-   **淡入动画** - 弹窗和消息的淡入效果
-   **滑入动画** - 消息从下方滑入
-   **打字动画** - AI 回复时的打字指示器
-   **弹跳动画** - 欢迎页面的机器人图标

### 响应式布局

-   **桌面端** - 左右分栏布局，左侧会话管理，右侧聊天区域
-   **移动端** - 自适应单栏布局，优化触摸操作

## 🔧 配置说明

### API 配置参数

-   **API Key** - DeepSeek 或其他兼容 OpenAI 格式的 API 密钥
-   **Base URL** - API 服务的基础 URL 地址
-   **Model** - 使用的 AI 模型名称（如：deepseek-chat）
-   **Temperature** - 控制回复随机性，范围 0-2，推荐 0.7

### 系统提示词类型

-   **default** - 默认对话风格（注意：包含特殊内容）
-   **coding** - 编程助手，适合技术问题
-   **creative** - 创意助手，适合创作和头脑风暴
-   **academic** - 学术助手，适合研究和分析
-   **casual** - 休闲助手，轻松友好的对话风格

## 📱 使用指南

### 首次使用

1. 打开应用后会自动创建一个默认对话
2. 点击右上角"⚙️ 配置"按钮设置 API 参数
3. 填入有效的 API Key 和其他配置
4. 点击"保存设置"完成配置

### 开始对话

1. 在底部输入框输入消息
2. 按 Enter 键或点击"SEND!"按钮发送
3. AI 会实时流式回复消息
4. 对话会自动保存到本地存储

### 管理会话

1. 点击"START NEW CHAT"创建新对话
2. 在左侧列表中点击切换不同会话
3. 鼠标悬停在会话上可以删除会话
4. 使用"🗑️ 清空对话记录"清除所有会话

## 🔒 安全说明

-   API 密钥仅存储在浏览器本地，不会上传到服务器
-   所有对话数据仅保存在本地浏览器存储中
-   建议定期备份重要对话内容
-   在公共设备上使用后建议清除浏览器数据

## 🐛 常见问题

### API 调用失败

-   检查 API Key 是否正确
-   确认 Base URL 地址是否可访问
-   检查网络连接是否正常
-   查看浏览器控制台的错误信息

### 对话记录丢失

-   检查浏览器是否清除了本地存储
-   确认没有使用无痕浏览模式
-   尝试刷新页面恢复数据

### 样式显示异常

-   确认浏览器支持现代 CSS 特性
-   检查是否有浏览器扩展影响样式
-   尝试清除浏览器缓存
